In html you have many different form input styles. This guide will show
you 5 different types with a submit button to submit your form.
A form: A html form is a container for different types of input elements, such as:
text fields, checkboxes, radio buttons, submit buttons, etc.

A form has an input and a label element the label defines what the input is
the input creates a type of form like a text field or a checkbox.
The for attribute of the label element should show for which input id
the label is defined.
