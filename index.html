Tables

Tables, tr th, td

click here for fullscreen
A table is a piece of data you can create with the 'table' element.
Within the table element you have the child elements: table rows (tr), table headers (th), and table data (td).
After you created the table element you create the table row
(as a child of the table parent) you do this by using 'tr' (table row element).
The table row will be a parent of the table header or the table data.
Usually you start with the table header and after creating the header you create the table date.
To use multiple rows or columns as 1 you have the attributes called colspan and rowspan.
They make the table spread out over multiple places depending on its value: colspan is horizontal and rowspan is vertical.
As you can see in the table below, the header fills in multiple areas (columns).
This is done by the use of the attribute colspan with a value of 11!

In the table below you see that the table header is bold and the table data is normal.
The table below represents a primary school calculation cheatsheet.
The horizontal data right from the x is multiplied by the vertical data beneeth the x.
Both horizontal next to the x and vertical below the x are table heads!
The data within (the outcome of the horizontal and vertical multiplication) is the table data!
